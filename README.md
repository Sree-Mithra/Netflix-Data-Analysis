# Netflix Data Analysis

This project performs exploratory data analysis on Netflix content to identify trends in genres, countries, and content growth.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data cleaning
- Movies vs TV Shows analysis
- Top countries producing content
- Genre distribution
- Netflix growth trends

## Dataset
Netflix Titles dataset from Kaggle.
